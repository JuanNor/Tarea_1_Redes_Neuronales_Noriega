# REPOSITORIO DE TAREA 1_RNA_JUANDEDIOS_NORIEGA
En este proyecto se implementa una red neuronal multicapa desde cero utilizando Python para reconocer dígitos escritos a mano utilizando un set de imágenes en MNIST.
El código fue obtenido de: https://github.com/MichalDanielDobrzanski/DeepLearningPython

Los archivos que se encuentran en esete repositorio son:
- network.py: Implementación de la red neuronal.
- entrenamiento.py: Entrenar al modelo.
- capturas (carpeta): Se pueden encontrar las capturas solicitadas.
- minst_loader.py: Necesario para el procesamiento de datos de entrenamiento.py
